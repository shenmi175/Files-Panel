Python + FastAPI Control Plane

文件与资源控制面板

管理当前节点的资源、文件、接入和日志。远程节点当前仅登记，不执行远程操作。

初次安装后可先通过 IP:端口 访问；接入域名后，服务会切回本地监听，由 nginx 负责对外入口。
