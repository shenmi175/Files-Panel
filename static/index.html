Guide

资源说明

当前 / 1m / 5m

当前表示最近一次采样值。

1m 和 5m 表示最近 1 分钟与 5 分钟窗口内的平滑结果，用来区分瞬时波动和持续压力。

趋势图

趋势图读取 SQLite 中持久化保存的采样点，不依赖页面是否持续打开。

范围越大，图表会做降采样，保证折线仍然可读。

CPU 与负载

CPU 利用率是最近一次采样周期内的使用率。

系统负载基于 Load，并按 CPU 核数归一化，两者相关但不等价。

网络与磁盘 I/O

网卡和磁盘分项显示当前瞬时速率。

它们更适合观察突发流量或 I/O 峰值，不用于表示长期占用率。

时间范围

15m、1h、6h、24h、7d 用于切换历史窗口。

页面会显示当前查看范围、采样周期和图表实际分辨率。
